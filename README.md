# 🛡️ Consent OS (IQCH)

**Consent OS** — это передовая автономная система управления конфиденциальностью и аудита цифровых разрешений. Проект предназначен для централизованного контроля за тем, какие сервисы имеют доступ к вашим персональным данным (Google OAuth и другие), анализа рисков с помощью ИИ и мгновенного отзыва небезопасных доступов.

## 🚀 Основные возможности (Прям всё-всё)

### 1. 📊 Интеллектуальный Dashboard
*   **Data Intelligence**: Визуализация всех активных подключений в реальном времени.
*   **Автономный ИИ-анализатор**: Система автоматически оценивает каждое разрешение по 100-балльной шкале риска на основе запрашиваемых скоупов (scopes).
*   **AI Insights**: Генерация персонализированных советов по безопасности (например, "Ваши контакты наиболее уязвимы, они используются в 5 сервисах").

### 2. 🕸️ Интерактивный граф связей (Data Graph)
*   **Визуализация узлов**: Центробежный граф на базе D3.js, показывающий связи между пользователем и сторонними приложениями.
*   **Цветовая индикация**: Мгновенное определение критических угроз (Красный — высокий риск, Желтый — средний, Зеленый — безопасно).
*   **Интерактивность**: Масштабирование, перетаскивание узлов и быстрый просмотр деталей сервиса при клике.

### 3. 🚨 Panic Mode (Lock Down)
*   **Мгновенная блокировка**: Кнопка "Lock Down High Risks" позволяет одним нажатием отозвать доступ у всех сервисов, чей рейтинг риска превышает 70%.
*   **Массовый отзыв**: Автоматизированный процесс прекращения сессий для критических угроз.

### 4. 🔍 OSINT и Deep Web Scan
*   **Поиск утечек**: Интеграция с инструментами OSINT (например, Holehe) для проверки, на каких сайтах зарегистрирован ваш email.
*   **Footprint Analysis**: Обнаружение цифровых следов, о которых вы могли забыть.

### 5. 📅 Risk Calendar & Audit Trail
*   **Календарь активности**: Тепловая карта (Heatmap) за последние 52 недели, показывающая дни наибольшей активности предоставления доступов.
*   **Логи аудита**: Детальная история всех действий: входы, сканирования, предоставление и отзыв прав.

### 6. 🔌 Расширение для браузера (Extension Bridge)
*   **Real-time Intercept**: Перехват новых OAuth-запросов прямо в браузере.
*   **Physical Revoke**: Помощь в физическом удалении приложения из настроек Google Account через автоматизацию.

## 🛠 Технологический стек

### Frontend
*   **React + Vite**: Молниеносная скорость работы интерфейса.
*   **Tailwind CSS**: Современный адаптивный дизайн с поддержкой Dark Mode.
*   **D3.js**: Сложная визуализация графов данных.
*   **Lucide React**: Набор минималистичных иконок.

### Backend
*   **FastAPI (Python)**: Высокопроизводительный асинхронный API.
*   **SQLAlchemy**: Мощная ORM для работы с базой данных.
*   **SQLite**: Легкая и быстрая БД для хранения локальных данных.
*   **Pydantic**: Строгая валидация всех входящих и исходящих данных.
*   **JWT Auth**: Безопасная авторизация пользователей.

## ⚙️ Установка и запуск

### 1. Подготовка Backend
```bash
cd consent-os/backend
pip install -r requirements.txt
python main.py
```
*Сервер запустится на `http://localhost:8001`*

### 2. Подготовка Frontend
```bash
cd consent-os/frontend
npm install
npm run dev
```
*Приложение будет доступно на `http://localhost:5173`*

## 🛡 Безопасность
Проект использует многоуровневую систему защиты:
1.  **Error Boundaries**: Защита интерфейса от "белого экрана" при сбоях.
2.  **Validation**: Проверка всех данных на стороне сервера и клиента.
3.  **Sanitization**: Очистка вводимых данных для предотвращения XSS/Injection атак.
